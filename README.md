# Github报刊订阅系统

这个GitHub报刊订阅系统是一个全面的订阅服务，
允许用户查看和订阅他们喜欢的刊物。
管理员和出版商可以管理用户的订阅和发布的刊物。
该项目的数据库需要运行一个初始化脚本，`dbs_init.sql`，
以配置必要的数据库结构和初始数据。

## 项目运行

首先，您需要在本地运行数据库，并执行数据库初始化脚本 `dbs_init.sql`。以下是如何在MySQL数据库中运行该脚本的步骤：

1. 安装MySQL数据库。如果你还没有安装，你可以从这里下载: [MySQL官方网站](https://www.mysql.com/)

2. 打开MySQL命令行工具，或者你喜欢的MySQL GUI工具（如MySQL Workbench）。

3. 在MySQL中运行`dbs_init.sql`文件：
    ```
    SOURCE path_to_your_dbs_init.sql;
    ```

## 功能

### 公共功能

所有用户可以享受以下功能：

- 更改自己的密码
- 查看（top）刊物
- 查看（top）出版商
- 查看top富哥
- 查看所有报刊

### 用户功能

注册用户可以享受以下额外功能：

- 查看自己的信息
- 更改用户名
- 更改地址
- 查看自己的订阅信息
- 续订报刊
- 订阅报刊  

### 管理员功能

管理员用户可以享受以下额外功能：

- 修改余额
- 重置密码
- 查看各个用户的订阅信息

### 出版商功能

出版商用户可以享受以下额外功能：

- 增加出版杂志
- 停刊
